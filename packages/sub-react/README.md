### 本地预览

可以直接通过主应用预览，微应用也可独立预览

- 主应用预览

```
主应用：http://localhost:7001/

主应用中的微应用：http://localhost:7001/child-sub-react
```

- 微应用独立预览

```
http://localhost:7002/
```
